## 下一步操作

### 1. 安装依赖（如果本地开发）

#### 前端
```bash
cd frontend
npm install
```

**注意**: 如果Node.js版本过低（当前v12.22.9），需要：
- 升级Node.js到20+版本，或
- 使用Docker运行前端（推荐）

#### 后端
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2. 启动Docker服务（推荐）

```bash
# 在项目根目录
docker-compose up -d

# 查看服务状态
docker-compose ps

# 查看日志
docker-compose logs -f
```

### 3. 访问服务

- **前端**: http://localhost:3000
- **后端API**: http://localhost:8000
- **API文档**: http://localhost:8000/docs
- **PostgreSQL**: localhost:5432
- **Redis**: localhost:6379

### 4. 初始化数据库

```bash
# 进入后端容器
docker-compose exec backend bash

# 安装Alembic（如果还没安装）
pip install alembic

# 初始化Alembic（首次运行）
alembic init alembic

# 创建迁移文件
alembic revision --autogenerate -m "Initial migration"

# 执行迁移
alembic upgrade head
```

## 常见问题

### Q: Node.js版本太低怎么办？
A: 使用Docker运行前端，或升级Node.js到20+版本。

### Q: 如何修改数据库配置？
A: 修改 `docker-compose.yml` 中的环境变量，或创建 `.env` 文件。

### Q: 如何添加新的API路由？
A: 在 `backend/app/api/` 目录下创建新的路由文件，并在 `main.py` 中注册。

### Q: 如何添加新的前端页面？
A: 在 `frontend/app/` 目录下创建新的页面文件（Next.js App Router）。

## 开发建议
